Cloud AI and ML Services

Definition of Terms:

Cloud: Cloud computing refers to the delivery of computing services (including servers, storage, databases, networking, software, analytics, and more) over the internet to offer faster innovation, flexible resources, and economies of scale.

Visualization Tools: Visualization tools are software applications or platforms that enable users to create visual representations of data. These tools help in turning data into understandable and actionable insights through charts, graphs, and interactive dashboards.

AWS (Amazon Web Services)

Definition: Amazon Web Services (AWS) is a comprehensive and widely adopted cloud platform, offering a diverse set of cloud services, including computing power, storage options, machine learning services, and more. AWS provides scalable and flexible solutions for businesses of all sizes.

Amazon SageMaker: Amazon SageMaker is a comprehensive and fully managed service that makes it easier for data scientists and developers to build, train, and deploy machine learning models. It offers a range of built-in algorithms, Jupyter notebooks, and model deployment capabilities.

Typical Uses: It is commonly used for building predictive models, image and text analysis, and time series forecasting.

Tools Used: Commonly used tools in AWS include TensorFlow, PyTorch, and Jupyter notebooks for machine learning development.

Amazon QuickSight: Amazon QuickSight is a cloud-powered business analytics service that empowers users to create interactive dashboards and reports. It offers a user-friendly interface for data exploration and visualization.

Typical Uses: It is typically used for data visualization, real-time monitoring, and sharing insights with stakeholders.

GCP (Google Cloud Platform)

Definition: Google Cloud Platform (GCP) is a suite of cloud computing services provided by Google. GCP offers a variety of services, including computing, storage, machine learning, and data analytics. It is known for its robust infrastructure and data analytics capabilities.

Google Cloud AI Platform: Google Cloud AI Platform is a comprehensive set of tools and services that enable data scientists and machine learning engineers to build, train, and deploy machine learning models. It provides a collaborative environment for data preparation, model development, and model deployment.

Typical Uses: It is typically used for machine learning model development, image recognition, and natural language processing.

Tools Used: Commonly used tools in GCP include TensorFlow, scikit-learn, and BigQuery for data analysis and machine learning.

Google Data Studio: Google Data Studio is a free and user-friendly data visualization tool that allows you to create customized, interactive dashboards and reports.

Typical Uses: It is typically used for creating interactive reports, data visualization, and sharing insights with stakeholders.

Azure (Microsoft Azure)

Definition: Microsoft Azure, commonly known as Azure, is a cloud computing platform and infrastructure provided by Microsoft. Azure offers a wide range of cloud services, including computing, analytics, storage, and machine learning. It is used by businesses for a variety of applications, from web hosting to machine learning.

Azure Machine Learning: Azure Machine Learning is an integrated, end-to-end data science and advanced analytics solution that simplifies the machine learning workflow. It offers tools and services for data preparation, model building, and deployment.

Typical Uses: It is typically used for building and deploying machine learning models, data exploration, and predictive analytics.

Tools Used: Commonly used tools in Azure include Microsoft's Azure Machine Learning Studio and Power Query for data transformation and analysis.

Azure Power BI: Power BI is a powerful business analytics service provided by Azure. It allows users to create interactive visual reports and dashboards, connecting to various data sources.

Typical Uses: It is typically used for creating interactive visual reports, data exploration, and sharing insights with stakeholders for data-driven decision-making.
